# 聪聪电影

### 项目简介

​	聪聪电影，可实现查看当前正在热映的电影和即将上映的电影的列表和详情，

​	有定位到当前城市功能和搜索电影名称功能，

​	可查看当前城市的电影院，有登录界面。
